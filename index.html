Get started

Sync data with a few lines of code in your Web application. Use our demo server or run your own.


Oulala, syncing data with the JavaScript client is a piece of cake. 
                var kinto = new Kinto({remote: "https://kinto.dev.mozaws.net/v1"});
                var tasks = kinto.collection("tasks");

                tasks.create({label: "Wake up early"});

                tasks.sync()
                  .then(function(res) {
                    console.log("Done!");
		  });

Running your own server is a matter of one command.

		> docker run -p 8888:8888 kinto/kinto-server
